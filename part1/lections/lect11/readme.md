#### task_01  

- исправить функцию sort_select()  
- должна работать правильно, но тут есть неточность:  

```py
def sort_select(arr):
    for n in range(len(arr), 0, -1):
        ind_max = 0
        for i in range(n):
            if arr[i] > arr[ind_max]:
                ind_max = i
        arr[ind_max], arr[-1] = arr[n-1], arr[ind_max]
    return arr
```
